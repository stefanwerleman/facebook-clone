# Instagram Clone

# Tools Utilized:
#### Front-end
* HTML/CSS/JavaScript
* CSS Skeleton

* NPM's **live-server** package
* JavaScript's **Document Object Model**

#### Back-end
* Server
   * **NodeJS** and **ExpressJS**
* database
   * **MongoDB** (No SQL)
